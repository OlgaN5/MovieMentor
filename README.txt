Авторизованные пользователи могут добавлять фильмы, которые они хотят посмотреть. Есть возможность изменить статус и добавить к фильму "похожие"
Далее на основании "похожих" к фильму любой пользователь может получить рекомендации к просмотру (анализируются «похожие» фильмы всех пользователей)

Использованы такие технологии как express, postgreeSQL, sequelize, bcrypt, jwt, swagger,mailgun
